# 🛒 Kasir Minimarket Java

Aplikasi desktop kasir berbasis **Java Swing** dan **MySQL** yang dibuat sebagai tugas akhir mata kuliah Pemrograman Desktop.

## ✨ Fitur

- Login Admin & Operator
- CRUD Data Barang
- CRUD Kategori
- CRUD Member
- CRUD User
- Transaksi Penjualan
- Perhitungan Otomatis
- Laporan Penjualan
- Validasi Input
- Password Hashing

## 🛠️ Teknologi

- Java
- Java Swing
- MySQL
- JDBC
- NetBeans IDE

## 🚀 Cara Menjalankan

1. Import project ke NetBeans.
2. Buat database MySQL.
3. Import file database.
4. Jalankan project.
